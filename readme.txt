Scan your WordPress site for broken images and fix them easily.

== Description ==

WP Broken Image Fixer scans posts, pages, and custom post types to find broken image URLs that can harm SEO and user experience.

The plugin provides a clean admin interface where scan results are displayed in a table with clickable post titles and image URLs.

== Features ==

* Scan posts, pages, and custom post types
* Detect broken image URLs
* AJAX-based scanning
* Persistent scan results
* Result count display
* Clean and simple admin UI
* Lightweight and fast

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the Plugins menu
3. Navigate to the Broken Image Fixer page
4. Click "Scan Now"

== Frequently Asked Questions ==

= Does it scan custom post types? =
Yes. All public, published post types are scanned.

= Does it fix images automatically? =
No. This version only detects broken images. Auto-fix is planned.

= Will it slow down my site? =
No. Scans run only when triggered from the admin panel.
